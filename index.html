Comandos do Git!

* Git status - Para visualizar o status atual do git

* Git log - para ver o histórico de versões no git

* Git add . - Para adicionar varios arquivos ao mesmo tempo ao stage

* Git branch - Para visualizar quais branch você tem e qual está ativa no momento

* Git reflog - Para visualizar o histórico de forma simplificada

* Git push -u origin master - Usar quando for a primeira vez que estiver enviando pelo git

* Git push - Para enviar um commit para o repositório no git hub

* Git commit -m 'descrição' - Para commitar (Preparar para enviar) a mudança que foram feitos nos arquivos

* Git remote add origin "Link do repositorio que quer enviar (Conectar)" - Para apontar ou fazer conexão com o repositorio de destino

* Git init" - Para inicializa o git no terminal

* Git add README.md" - Para criar um README no github

* Git branch -M main" - Renomar a branch principal

* Git reset --hard 'id da versão" - Para voltar a uma versão anterior
